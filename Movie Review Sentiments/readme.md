Description
Developed a binary sentiment analysis model for IMDB movie reviews using Natural Language Processing (NLP). 
The project employs CountVectorizer for text vectorization and compares Gaussian, Multinomial, 
and Bernoulli Naive Bayes classifiers to identify the most effective approach for sentiment classification.

Key Highlights
NLP-based text preprocessing
Bag-of-Words feature extraction
Naive Bayes model comparison
Accuracy-based performance evaluation
Positive/Negative sentiment prediction
